ArrayList源码中一段很迷的代码

2018.9.25 14:02 by ovo

这是在使用add(int  index,object e)的时候发现的问题，你即使设定了ArrayList的大小，也依然无法直接插入add(1,e)，原因就是在这个add方法中，会有一次判断，这次判断会比较Index和size的值，index必须等于或小于size，负责会报错。

源代码

/** * A version of rangeCheck used by add and addAll. */

private void rangeCheckForAdd(int index) {

  if (index > size || index < 0)

    throw new IndexOutOfBoundsException(outOfBoundsMsg(index));

}
